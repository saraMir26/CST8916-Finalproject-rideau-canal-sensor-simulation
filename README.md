# Rideau Canal Sensor Simulation

## Overview

This project simulates IoT sensors for the Rideau Canal Skateway monitoring system. It generates real-time environmental data from three locations and sends it to Azure IoT Hub.

The simulator mimics real-world sensor behavior by producing data every 10 seconds.

### Locations Simulated
- Dow's Lake
- Fifth Avenue
- NAC (National Arts Centre)

---

## Technologies Used

- Python
- Azure IoT Device SDK
- python-dotenv

---

## Prerequisites

- Python 3.x installed
- Azure IoT Hub with registered devices
- Device connection strings
